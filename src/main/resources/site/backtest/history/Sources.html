Format Provider Type Instrument
${source.format} ${source.provider} ${source.instrument.type} ${source.instrument.name}

There are two Formats available:

  • PCF (Price Candle Format) - provides minute level price candles, and is very fast to test against any time scale.
  • PTF (Price Tick Format) - provides tick level price candles, for maximum accuracy but is significantly slower to test with.
